## 第一梯队：刚需 + 高频 + 没人做好

### 1. "对接不求人"系列 — 集成 Skill ✅ 已选

开发者最恨的事：花 3 天读文档对接一个 SDK，还踩坑。

- 微信支付/支付宝对接 skill（签名、回调、退款、对账全套）
- OAuth2 SSO 对接 skill（飞书、钉钉、企业微信、GitHub）
- 短信/邮件/推送 skill（阿里云/腾讯云/极光）
- OSS 对接 skill（上传、分片、签名 URL、CDN）

**为什么能卖**：每个项目都要做，每次都踩坑，文档永远过时。一个 skill 搞定 = 省一个开发周。

### 2. "造轮子终结者"系列 — 常见功能 Skill

重复实现的东西，做到极致。

- 权限系统 skill（RBAC + 数据权限 + 字段级控制）
- 多租户 SaaS 架构 skill（租户隔离、数据路由、计费）
- 实时协作 skill（CRDT/OT、光标同步、冲突解决）
- 审批工作流 skill（流程引擎、节点配置、回调通知）

### 3. "救命系列" — 生产环境诊断 Skill

凌晨 3 点线上挂了，客户在催。

- "服务启动失败"自动诊断（读日志 → 分析异常 → 给修复方案）
- "数据库慢查询"分析 skill（抓 SQL → explain → 建索引）
- "内存泄漏"排查 skill（dump 分析 → 定位泄漏源）
- "接口变慢"性能诊断 skill（链路追踪 → 瓶颈定位）

**为什么能卖**：紧急程度高 = 付费意愿强。按次收费 9.9~49 元，开发者毫不犹豫。

## 首发产品定义：微信生态全链路 Skill

> 用户在小程序下单 → 公众号收到通知 → 微信支付 → 模板消息推送发货 → 企业微信客服跟进

**核心价值**：一个 skill 串通整个微信用户生命周期，UnionID 统一身份。

### 包含模块

| 模块 | 内容 | 单独做的痛点 |
|------|------|-------------|
| **小程序登录** | code2Session、手机号快速验证、UnionID 统一 | 文档分散，session_key 刷新机制容易搞错 |
| **公众号网页授权** | OAuth2 授权、用户信息、关注状态检测 | scope 区分、回调域名配置坑多 |
| **UnionID 打通** | 小程序 ↔ 公众号 ↔ 开放平台用户统一 | 需要开放平台绑定，很多人搞不清 |
| **模板/订阅消息** | 申请模板、下发、回调确认 | 每种消息类型限制不同，一次性订阅 vs 长期订阅 |
| **微信支付** | JSAPI/H5/小程序支付、退款、对账 | 证书配置、签名、回调幂等 |
| **客服消息** | 接收消息、自动回复、转人工 | XML 解析、加密解密、5秒响应限制 |

### 边界

**做**：后端集成层 — 统一 token 管理 + 用户身份打通 + 消息推送 + 支付回调 → 开箱即用的 Spring Boot starter

**不做**：微信支付 SDK 本身（用 IJPay 等成熟方案）、小程序前端组件（不在 skill 范围）
